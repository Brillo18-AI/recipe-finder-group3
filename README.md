# Recipe Finder (Group 3)

## Project Description
Recipe Finder is a web application built with Angular that allows users to search for recipes using ingredients and view results with images.

## Features
- Search recipes by ingredient
- Display recipe results with images
- Clean UI using Angular Material
- MVVM architecture using services

## Tech Stack
- Angular + TypeScript
- Angular Material
- MVVM Pattern
- External Recipe API
